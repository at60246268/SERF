Patrones de Diseño Aplicados

Al generar un reporte, el sistema ejecuta la siguiente secuencia:

0 PROXY

ReporteProxy valida credenciales y roles antes de permitir acceso al reporte

1 SINGLETON

ConfiguracionGlobal proporciona tasas de cambio únicas para conversión de monedas

2 PROTOTYPE

PlantillaReporte clona la plantilla según el tipo seleccionado (Mensual/Trimestral/Anual)

3 BUILDER

ReporteBuilder construye el reporte paso a paso con un API fluido

4 COMPOSITE

ComponenteReporte organiza el contenido jerárquicamente con secciones y elementos

5 DECORATOR

MarcaAguaDecorator y FirmaDigitalDecorator agregan seguridad dinámicamente

6 FACADE

ReporteFinancieroFacade simplifica la coordinación de todos los patrones anteriores
